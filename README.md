## Spending Control
 
The spending control platform aims to help people control their monthly expenses.
 
## Technology 
 
Here are the technologies used in this project.
 
* Javascript
* PHP 7.4.8
* MariaDB 10.4.14
* HTML5
* CSS3
* PHPMAILER 6.1.7
